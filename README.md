# 摩斯电码

> 摩斯电码编解码工具，文字摩斯互转，音频播放，速度调节

## 功能概览

- **文字到摩斯码：中文支持拼音转换，英文大小写通吃**
- **摩斯码到文字：解码为字母和数字**
- **音频播放：点划信号音，速度可调（WPM 值调节）**
- **闪灯模拟：同步闪烁指示信号节奏**
- **标准 ITU 摩斯码表对照显示**

## 操作说明

1. 打开 index.html（浏览器直接运行，无需服务器）
2. 按界面引导输入/操作
3. 结果实时预览，支持一键复制导出

## 技术栈

- **HTML5**：语义化结构
- **CSS3**：白色极简主题，Flex/Grid 布局
- **Vanilla JavaScript**：纯原生代码，零依赖
- **ES6+**：sync/await、箭头函数、模板字符串
- **localStorage**：偏好设置持久化
